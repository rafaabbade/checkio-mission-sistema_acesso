Enunciado

Um prédio comercial implementou um novo sistema de segurança que restringe o acesso a certos horários. O sistema deve permitir o acesso apenas durante o horário comercial (9 às 18) e para pessoas com credenciais válidas. Escreva uma função que determine se o acesso de uma pessoa deve ser permitido com base no horário e no status de suas credenciais.

Entrada: hora (inteiro, 0-23), credenciais_validas (booleano).

Saída: Uma string "Acesso Permitido" ou "Acesso Negado".
